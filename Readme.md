650 Applied ML, LLM, and Agentic AI Case Studies

This repository contains a curated dataset of 650 real-world ML, LLM, and Agentic AI use cases from leading companies across industries.

Each row captures how organizations apply AI in production to solve concrete business or engineering problems.

Dataset

File: 650 ML and LLM use cases.csv

Columns:

Company – Organization using AI

Industry – Business domain

Short Description – Problem summary (≤ 5 words)

Title – Case study or technical write-up title

What You’ll Find

RAG and AI agent implementations

Recommendation and search systems

Prompt and context engineering

Developer productivity with LLMs

Code review, migration, and knowledge systems

Usage:

import pandas as pd
df = pd.read_csv("650 ML and LLM use cases.csv")


Use it for research, inspiration, benchmarking, or trend analysis.

Purpose:

A practical reference of how AI is actually used in production, not theory.
