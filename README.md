650 Applied ML, LLM, and Agentic AI Case Studies

This repository contains a curated dataset of 650 real-world Machine Learning (ML), Large Language Model (LLM), and Agentic AI case studies implemented by leading global companies across industries.

The dataset focuses on practical, production-level applications rather than theoretical examples. Each row represents how a company used AI to solve a concrete business or engineering problem.

Dataset Overview

File:
650 ML and LLM use cases.csv

Each entry captures:

The company applying AI

The industry context

A short problem summary

A detailed case study title or reference

This dataset is intended for engineers, data scientists, AI architects, product leaders, and researchers who want real examples of how modern AI systems are being deployed.

Columns Explained
Column Name	Description
Company	Organization that implemented the ML / LLM solution
Industry	Business domain (e.g., Fintech, Social Platforms, Tech, E-commerce)
Short Description (< 5 words)	High-level summary of the problem being solved
Title	Detailed title of the case study, blog, or technical write-up
